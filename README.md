# 基金分析器

输入基金代码 → 全量分析数据

## 数据来源
- 东方财富（净值、估值、排行）
- 新浪财经（ETF K线）

## 分析功能
- ✅ 实时净值/涨跌幅
- ✅ 阶段回报（1月/3月/6月/1年/2年/3年/年初至今）
- ✅ 风险指标（波动率/最大回撤/夏普/胜率/盈亏比）
- ✅ 均线分析（MA10/MA60/乖离率/金叉死叉）
- ✅ 估值评级（PE/PB分位）
- ✅ 交易信号（买入/持有/卖出/观望）
- ✅ 净值走势图（含均线）
- ✅ 快速查询（预设基金）

## 构建

```bash
flutter pub get
flutter build apk --release
```

## GitHub Actions

推送代码到 main 分支自动触发云编译，在 Actions 页面下载 APK。
